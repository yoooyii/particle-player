# 🎧✨ Particle Player — 粒子音乐互动页

一个用 **p5.js** 做的网页小玩具：  
点击页面撒粒子 ✨，播放音乐 🎵

> 支持 YouTube / Spotify / SoundCloud / 网易云 / Bilibili 外链播放

---

## 🌟 在线体验
👉 [点击这里试玩](https://yoooyii.github.io/particle-player/)

---

## 🪄 功能介绍

- 🎶 支持外链播放（YouTube / Spotify / SoundCloud / 网易云 / Bilibili）  
- 💬 自定义词库：可添加你自己的文字、emoji、句子  
- 📂 可导入 / 导出 JSON 保存你的词库  
- 🧩 轻盈的粒子物理效果，随机飞舞  
- 🔮 简洁界面 + 模糊玻璃风格  

---

## 🖱️ 操作说明

| 动作 | 功能 |
|------|------|
| 点击页面 | 撒粒子 ✨ |
| 按 **C** | 清屏 |
| 按 **T** | 开关文字显示 |
| 点击播放键 | 播放当前选中歌曲 |
| 输入链接并点击 “＋ 添加链接” | 添加你自己的音乐外链 |
| 在左上方的面板中输入文字后点“应用并保存” | 自定义飘动文字 |

---

## 📦 文件说明

| 文件 | 作用 |
|------|------|
| `index.html` | 主网页，包含播放器和粒子界面 |
| `sketch.js` | 粒子逻辑与交互控制 |
| `songs.js` | 预设的音乐外链列表 |
| `words.js` | 默认词库（可自定义） |

---

## 🧠 技术栈
- [p5.js](https://p5js.org/) — 用于粒子动画和画布渲染  
- HTML + CSS — UI 与播放器布局  
- Vanilla JS — DOM 控制与外链解析  

---

## ⚠️ 注意
- 音乐外链均来自公开平台，不存储任何音频文件。  
- 若涉及版权问题，请在合理范围内使用。  
- GitHub Pages 用户访问地区不同，部分平台（如 Bilibili、网易云或者Youtube、Spotify）可能无法正常播放。
